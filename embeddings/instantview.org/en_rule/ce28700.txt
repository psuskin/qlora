This is the description of the functionality of the class "CX_STRUCTURE_MONITOR" regarding Description: This pseudo class corresponds to the class CX_ACCOUNT. While the class CX_ACCOUNT is used to monitor business objects, this pseudo class CX_STRUCTURE_MONITOR is used to monitor structure nodes of an evaluation group tree. An example of this is the storage of planned costs in structures/groups of cost elements.
