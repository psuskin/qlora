This is the description of the class "Management of a date period with a string field": The following program code shows the definition of a window through which a date period can be input and output. A surface object String is used, whose content is placed into the object with the command DrainWindow and filled with FillWindow. Window(Win, 50, 70, 410, 37, T("Beispiel Nr. 1", "Example No. 1")) [ Var(sdate) INITIALIZE: CreateTransObject(CX_SPAN_DATE) -> sdate "2Q04 - 4Q05" sdate Put sdate FillWindow ] { Prompt(10, 10, T("Datumsperiode:", "span-date:")) String(CX_SPAN_DATE::this, 150, 10, 235) }.
