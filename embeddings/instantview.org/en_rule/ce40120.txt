This is the description of the functionality of the class "CX_SYSTEM_TEXT_LITERAL" regarding Description: This class represents the text constants from the ClassiX® system. This concerns, for example, the names of all units of measurement and currencies, names of months, days of the week, terms such as calendar week, quarter and all error messages. See also: Text constants in the ClassiX® system. The class provides the interface to develop viewers and editors for these texts with InstantView®, supporting the extension with any other languages. All CX_SYSTEM_TEXT_LITERAL objects are obtained by calling the method GetLiterals of the System object.
