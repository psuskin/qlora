This is the description of the functionality of the class "CX_PHRASE*" regarding Description (Business Pattern): This class is used to model words. It is closely related to the CX_SIGNIFICATION class for modelling meaning. The class CX_PHRASE is derived from the class CX_STIMULUS, which represents the role "stimulus" of a physical or chemical phenomenon. Thus, CX_PHRASE is to be regarded as an "acoustic stimulus" perceived by a system (or a person).
