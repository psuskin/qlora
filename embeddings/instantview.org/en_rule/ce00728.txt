This is the description of the class "CX_STRUCTURE - Processing": The functions grouped together in this section provide as a result the added return values for a specific function of all objects in a structure. Function name Description DeepAddVoid Adds the results of a specific function without parameters for all subordinate elements DeepAddObj Adds for all subordinate elements the results of a certain function with an object as parameter DeepAddObjIntObj Adds for all subordinate elements the results of a certain function with integer and two objects as parameters DeepAddObjIntIntObj Adds for all subordinate elements the results of a certain function with two integers and two objects as parameters.
