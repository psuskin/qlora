This is the description of the functionality of the class "CXP_PRINT_JOB - Printer job" regarding Description: This class represents a print job in System. A CXP_PRINT_JOB object should be created before document preparation using the document method CreatePrinterJob. If the document has several page settings, further print jobs are created automatically, one print job per page setting. If spooling is switched off, the print job should be started after the document is ready using the StartPrinting method. If spooling is switched on, the print job is started automatically as soon as the number of completed pages reaches the number of spool pages.
