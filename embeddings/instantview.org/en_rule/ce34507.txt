This is the description of the class "CX_INDEX": Attention: This class is obsolete and should no longer be used. Please use CX_OBJECT_DICTIONARY instead. This is the description of the functionality of the class "CX_INDEX" regarding Description: The class CX_INDEX corresponds largely to the class CX_DICTIONARY. The difference is that it does not support multiple languages. CX_INDEX objects are suitable for registering objects under terms that do not depend on the language currently selected (key numbers, company names, .). CX_INDEX belongs to the family of dictionary classes.
