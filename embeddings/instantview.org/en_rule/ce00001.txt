This is the description of the class "Working with this documentation": This documentation is divided into the following sections: Introduction to the system Description of the classes Class reference The first section provides a rough overview of the system and its internal structure. The description of the classes is aimed both at new users who want to get to know the functionality and at more experienced users who use it as a reference book. Basically, each class is not only described for itself, but related classes are also shown by an inheritance hierarchy and possible uses are illustrated by examples of use and cross-references to other classes. Finally, the class reference is aimed at the experienced user who is looking for information for a specific class.
