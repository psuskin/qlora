This is the description of the functionality of the class "CX_DESK_PHONE* - Fixed line numbers" regarding Description: An object of the type CX_DESK_PHONE contains a telephone number of a fixed network telephone. It establishes the connection to a real object via a fixed network number. A code example is located at CX_PHONE. For information on the method directory (MDI) and data directory (DDI), please refer to the class documentation.
