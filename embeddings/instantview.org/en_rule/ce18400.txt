This is the description of the class "CX_DICTIONARY": Attention: This class is obsolete and should no longer be used. Please use CX_OBJECT_DICTIONARY_ML instead. This is the description of the functionality of the class "CX_DICTIONARY" regarding Description: The CX_DICTIONARY class allows objects to be registered under any multilingual designations. An object can have any number of identifiers, and any number of objects can be registered in the dictionary under one identifier. The objects can be retrieved very quickly by means of the names assigned to them. The dictionary also manages a back-reference, so that you can obtain all the names under which an object was entered. When searching the dictionary, a distinction is made between upper and lower case letters (case sensitive). CX_DICTIONARY belongs to the family of dictionary classes.
