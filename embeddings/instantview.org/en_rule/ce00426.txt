This is the description of the functionality of the class "Managing a formula condition with string fields" regarding Description: The following program code shows the definition of a window through which a formula condition can be input and output. Two interface objects String are used. To display or modify the object, the content of its field can be placed into the object with the command DrainWindow and filled with FillWindow. In contrast, the formula must be determined explicitly from the formula condition or be placed in it, since it behaves externally like its object.
