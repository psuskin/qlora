This is the description of the class "CX_SPAN_DATE - Start, end and duration": The functions grouped together in this section change the limits and/or duration of a date period. Function name Description SetStartOfPeriod Setting a start date with changing the period SetEndOfPeriod Setting the end date with changing the period MovePeriodToStartDate Setting the start date without changing the period MovePeriodToEndDate Setting the end date without changing the period SetPeriodBorders Setting the start and end date SetDuration Setting the period duration.
