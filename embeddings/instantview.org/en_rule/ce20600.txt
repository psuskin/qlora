This is the description of the functionality of the class "CX_ACCOUNT - Account (1-dimensional monitors)" regarding Description: This class manages an account. Accounts are programmed which slots they should add up. They must be slots of type CX_AMOUNT, since they are to be addable values. It is also possible to specify a CX_AMOUNT to define the format of the totals to be stored (automatic conversion and formatting of the held values) For example, currency for a specific order. Fibu account or the unit of measure for a warehouse account.
