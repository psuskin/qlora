This is the description of the functionality of the class "CX_KEYWORD" regarding Description: CX_KEYWORD is an auxiliary class for working with entries of objects of classes CX_DICTIONARY, CX_DICTIONARY_CI, CX_INDEX and CX_INDEX_CI. CX_KEYWORD objects cannot be created with CreateTransObject or CreatePersObject, but are only the result of functions of the above mentioned classes. The transient objects of class CX_KEYWORD combine the multilingual identifier and the designated object.
