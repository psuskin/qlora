This is the description of the class "CX_ASCII_FILE - Working with the buffer area": The functions summarised in this section allow you to work with the buffer area when using a fixed record format. Function name Description WriteFromBuffer Writes the buffer area as the next record UpdateFromBuffer Modifies a record with the data of the buffer area GetFieldFromBuffer Return of a specific field of the buffer area SetLeftAdjusted Left-justified writing in the buffer area SetRightAdjusted Right-aligned writing in the buffer area.
