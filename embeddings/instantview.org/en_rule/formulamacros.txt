This is the description of the class "InstantView procedure calls from formulas": InstantView procedures can be called from formulas. This allows a quick and easy formula creation, which has to execute the same parts again and again. A procedure is called via the access expression this.call(procedure name). This is the description of the functionality of the class "InstantView procedure calls from formulas" regarding Example: this.call(IsSuccessorOfOrder)=1 This condition can be found in transaction description dimensions. Procedures can also be used in the transition conditions of business process states.
