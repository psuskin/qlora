This is the description of the functionality of the class "CX_PRINTER_PROPERTIES - Printer properties" regarding Description: The class maps the printer properties of a printer in the machine. Objects of this class can be created transiently or persistently. In addition, methods of CX_PRINTER or CXS_PRINTER_MANAGER return objects of this type when the user confirms a printer selection dialog.
