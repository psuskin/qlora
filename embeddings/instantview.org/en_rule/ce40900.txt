This is the description of the functionality of the class "CXS_REP_COLLECTION" regarding Description: This class represents a root entry point collection of the database and encapsulates the actual COLLECTION. The REP-Collections can be accessed in InstantView via the domain to which they belong. To avoid inconsistencies, the DDI fields "name", "flag", "type" and "segment" should be read-only once the object refers to a valid REP collection. (Either after CreateREP has been called or if the object comes from a CXS_DOMAIN ).
