This is the description of the class "Elementary data types": Elementary data types are special types whose main task is to organise the storage of whole numbers and texts in the database with maximum efficiency. These data types represent the basic data structures that can be directly processed by any computer or programming language. Elementary data types are in contrast to objects of classes or collections. The following elementary data types are available in ClassiX: Data type Data type name Integer INTEGER Text STRING Multilingual text CXB_MULTIPLE_STRING.
