This is the description of the functionality of the class "CX_CONDITIONED_ATTRIBUTE - Conditional attribute" regarding Description: A conditional attribute allows the content of a dynamic data field to be filled by a conditional object table. It also provides a more detailed description of this field. An example is the definition of an article whose colour is determined depending on its height. So the following rules could apply: The conditions can be values of other attributes, which makes a dependence of the value of this CONDITIONED_ATTRIBUTE attribute on the value of others: e.g. if mounting point = right or = both sides then length right = length; if mounting point = left or = both sides then length left = length.
