This is the description of the class "CX_ASCII_FILE - File format and status": The functions summarized in this section are used to specify the file structure and status of the file. Function name Description SetVarFileFormat Setting a variable record format SetFixFileFormat Setting a fixed record format ComputeRecordLength Determining the record length ContainsStrings Determine string processing in the file Exists Check whether a specific file exists Open Opening a file ForcedOpen Definitive opening of a file ReOpen Repeated opening of a file Close Closing a file.
