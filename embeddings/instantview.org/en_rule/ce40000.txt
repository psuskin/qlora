This is the description of the functionality of the class "CX_MSG_RECEIVER" regarding Description: Objects of this class represent the message receivers(receivers) in a CX_MESSAGE_INFO object. The data field next contains the implementation of this message from the base module, if the message was implemented in a base module. next is only set for messages received in the module, not for messages received from widgets. Several receiver modules can be chained via the data field next (chained list).
