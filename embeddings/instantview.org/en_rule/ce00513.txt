This is the description of the class "CX_TIMED_TRIGGER - Modification": The functions grouped together in this area change a specific signal object. Function name Description ResetTriggered Set the number of signals already sent to 0 SetAction Setting the function to be performed SetMsg Setting the message to be sent SetObject Set the object for which the function is to be performed SetActionTerm Setting the description of a point in time / points in time SetTimedOut Setting a timeout.
