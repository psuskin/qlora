This is the description of the functionality of the class "Management of a value with two surface objects" regarding Description: The following program code shows the definition of a window through which a value can be input and output. Either String or Numeric is used to manage the numerical part of the value. To represent the unit, either String, Combo or Enum is used. With Combo it is possible to preassign units, but others can also be selected (manually). With an enum , however, a function or a file provides a table with the units and their assigned values, so that no other units can be entered.
