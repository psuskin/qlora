This is the description of the functionality of the class "CXS_DATABASE" regarding Description: This class represents a database. (See also CXS_DATABASE and CXS_SEGMENT) The database path in the DDI field "path" normally corresponds to the environment variable where the actual path is stored. To get the full database path, always use GetPath or GetOSPath.
