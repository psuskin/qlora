This is the description of the functionality of the class "CX_QUERY_EXPR" regarding Description: Objects of this class are returned by CX_QUERY::Syntax and represent the syntax tree of the query. Via String you get an identifier that describes the node type and the part of the query string it represents. With next you get the child nodes.
