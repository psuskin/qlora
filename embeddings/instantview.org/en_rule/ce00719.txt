This is the description of the class "CX_STRUCTURE - Insert": The functions summarised in this section allow the insertion of new objects into a structure. Function name Description InsertFirst Inserts the element first in the list of child elements InsertLast Inserts the element last in the list of subordinate elements InsertBefore Inserts the element before a specific element InsertAfter Inserts the element after a specific element InsertLevelAbove Insert a new layer above the layer InsertLevelBelow Insert a new layer between the layer and subordinate elements.
