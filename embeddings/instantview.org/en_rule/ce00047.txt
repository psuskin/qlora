This is the description of the class "CXB_MULTIPLE_STRING - Comparisons": The following overviews show the comparison operators and the operands with which a multilingual text can be linked. All operators return the result of the comparison as a truth value. Equality Larger Smaller Preliminary remark: When a multilingual text is compared with another (multilingual) text, only the text in the currently set national language is compared with the other text.
