Division CX_FLOAT is the only arithmetic type based on the technical double type and therefore also supports division by 0 without errors. A division by 0 returns ∞ (infinite), which can be used for normal calculations. The division 0/0 and the multiplication 0 * ∞ returns NaN (not a number), which again can be used for calculation, but this error type returns NaN with every other operation. These special values can be queried via IsInfinity() and IsNaN().
