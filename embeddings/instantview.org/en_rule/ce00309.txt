This is the description of the class "CX_PERIODIC_SPAN_TIME - Start and end": The functions grouped together in this section redefine the limits of a time specification. Function name Description SetStartOfPeriod Setting the start time with changing the period SetEndOfPeriod Setting the end time with changing the period MovePeriodToStartTime Setting the start time without changing the period MovePeriodToEndTime Setting the start time without changing the period SetPeriodBorders Setting the start and end time.
