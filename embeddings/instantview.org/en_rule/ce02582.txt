This is the description of the class "Definitions for date periods": The following table shows all available restriction types according to which the date period can be defined. The respective value is recorded in the lower 2 bits of the validEnum field. It depends on the respective value whether the start date and the period length are relevant and which additional date types can be selected. See: (Semi) Unlimited Intervals. Definitions for date periods Definition Constant RealStartOfPeriod RealEndOfPeriod Meaning (start) date infinite (unlimited) ENDLESS NULL NULL none semi-restricted, no start date TO NULL EndDate semi-restricted, no end date FROM NULL Start date limited FROM_TO Duration > 0: Start- otherwise end date.
