This is the description of the functionality of the class "CX_AUDIT_MANAGER" regarding Description: This object represents the audit manager. It exists only once in the system, and its object can be retrieved using the InstantView® command GetManager(AUDIT). In contrast to debuggers, the audit manager is used to follow the program flow at the logical level of the application. Trace statements in the InstantView® code identify the steps of the application. For each section marked with Trace, the objects changed in the process are shown.
