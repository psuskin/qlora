This is the description of the functionality of the class "CX_JSON_ARRAY" regarding Description: This class represents JSON arrays and, like CX_JSON_OBJECT, offers the possibility to access the elements of the array with access expressions. In addition, the class offers array-specific methods that simplify iterating over the contents. 226560, iterate and cardinality are also supported. 232555 is also insert supported. The development of this class is not yet finished and the behaviour may change in future versions.
