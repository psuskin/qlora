This is the description of the functionality of the class "CX_PAGE_PROPERTIES - Page properties" regarding Description: This class represents the page format of documents in this machine. The instances of this class can be made persistent using the CreatePersObject method. Page settings of a document are set with e.g. SetPagePropertiesAndPrinter. The default values of a printer can be queried with GetDefaultPageProperties.
