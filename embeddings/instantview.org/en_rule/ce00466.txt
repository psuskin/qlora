This is the description of the class "CX_SYSTEM_OBJECT - Other": The functions grouped together in this area cannot be assigned to one of the other areas. Function name Description DerivedClasses Return of all classes derived from a given class GetEnvironment Returns the value of an operating system variable GetExtDDI Returns the DDI for a class GetExtMDI Delivers the MDI for a class GetSlotEntries Return of all existing slots GetSlotEntry Return of a specific slot GetSlotNames Return of all slot names Wait Pauses the programme run for a certain time.
