This is the description of the class "CX_DATE - Multilingual text": The functions summarised in this section provide multilingual text output for a specific date. Function name Brief description MLWeekdayNameOfDay multilingual weekday name MLShortWeekdayNameOfDay multilingual abbreviated weekday name MLNameOfMonth multilingual name of the month MLShortNameOfMonth multilingual abbreviated name of the month MLNameOfDay multilingual (holiday) name of the day MLSpecialNameOfDay specific multilingual (holiday) name of the day.
