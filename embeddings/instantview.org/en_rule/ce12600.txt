This is the description of the functionality of the class "CX_SLOT_ATTRIBUTE - Preset attribute or attribute" regarding Description: A preset attribute is used to further specify a slot (dynamic data field). Such an attribute describes an item in more detail. The item's data fields are represented by slots. CX_SLOT_ATTRIBUTE is of the type of a so-called value attribute, i.e. it directly represents a value (function IsValueAttribute) that can be edited (function Value). Examples:.
