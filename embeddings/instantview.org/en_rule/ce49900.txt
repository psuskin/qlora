This is the description of the functionality of the class "CX_MULTIPLE_WORD_XML" regarding Description: This class is a container for multilingual XML documents. A separate XML document is kept for each language. This object offers all functions of the class CX_WORD_XML (greyed out). If a function of CX_WORD_XML is called, the XML document of the currently set language is loaded first and then the function is called on it. If a language is set as INTEGER parameter, then (-1) can be specified to select the currently set system language. Before CX_WORD_XML functions can be called directly on a CX_MULTIPLE_WORD_XML, a CX_WORD_XML object must be assigned to the object for the system language, otherwise the call would generate an error message.
