This is the description of the functionality of the class "CX_MASTER_LOCALE from DLL version 4.5.2.173619" regarding Description:(Business Pattern): The class serves as a container that holds all locale information known in the system: at first level in Collection locales of all objects of type CX_COUNTRY_LOCALE. All read-in locales - see CX_SYSTEM_OBJECT::GetAllLocales - are obtained as transient CX_MASTER_LOCALE object. The destructor releases the memory of all subordinate objects. With method StoreXML the whole locale information is saved as XML. Code example: * MA = Member Access Function * I = Indexable data field.
