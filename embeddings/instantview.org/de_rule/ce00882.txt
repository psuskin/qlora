Dies ist die Beschreibung der Klasse "Modi zum Öffnen einer Datei": Zum Öffnen einer Datei stehen folgende Modi zu Verfügung, die an die Syntax der Modi der Programmiersprache "C" angelehnt sind. Dabei werden die Dateien allerdings immer im binären Modus geöffnet. Modus Beschreibung r oder rb Öffnet eine binäre Datei im Lesemodus. w oder wb Öffnet eine binäre Datei im Schreibmodus. Falls sie schon existiert, werden vorhandene Inhalte zerstört, ansonsten wird sie neu angelegt. Sollte eine Datei, für die eine Index-Datei existiert, in diesem Modus geöffnet werden, so wird die Index-Datei gelöscht, da die eigentliche Datei neu angelegt wird. a oder ab Öffnet eine binäre Datei im Erweiterungsmodus. D.h., dass die Datei nur erweitert wird und keine bestehenden Daten überschrieben werden (können). Existiert die Datei noch nicht, so wird sie neu angelegt. r+, rb+ oder r+b Öffnet eine binäre Datei zum Lesen und Schreiben. w+, wb+ oder w+b Öffnet eine binäre Datei zum Lesen und Schreiben. Existiert die Datei noch nicht, so wird sie neu angelegt, ansonsten werden bestehende Daten gelöscht. Sollte eine Datei, für die eine Index-Datei existiert, in diesem Modus geöffnet werden, so wird die Index-Datei gelöscht, da die eigentliche Datei neu angelegt wird. a+, ab+ oder a+b Öffnet eine binäre Datei zum Lesen und Schreiben. Beim Schreiben wird die Datei erweitert, so dass bestehende Daten nicht überschrieben werden (können). Sollte die Datei beim Öffnen nicht existieren, so wird sie neu angelegt.
