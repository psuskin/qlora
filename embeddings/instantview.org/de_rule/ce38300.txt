Dies ist die Beschreibung der Funktionalität der Klasse "CX_INTEGER" bezüglich Beschreibung: Mit dieser Klasse wird eine ganze Zahl gespeichert. Dabei muss die Zahl aus dem Wertebereich -2.147.483.648 (= - 231) bis 2.147.483.647 (= 231-1) stammen. Von Objekten der Klasse CX_INTEGER sind Daten vom Typ INTEGER zu unterscheiden: CX_INTEGER: INTEGER: 199051 Wird auf einem INTEGER Wert auf dem Stack eine Methode per Call ausgeführt, dann wird der INTEGER für den Aufruf implizit in ein temporäres CX_INTEGER-Objekt umgewandelt, auf welchem dann der Aufruf stattfindet. So können alle hier aufgeführten Methoden auf normalen INTEGER-Zahlen ausgeführt werden. Dabei sollte beachtet werden, dass das Objekt nur für die Dauer des Aufrufs existiert und Methoden, die das Objekt verändern (Bsp: DivAssign) nur dieses temporäre Objekt verändern und somit keine Auswirkung auf den INTEGER-Wert haben.
