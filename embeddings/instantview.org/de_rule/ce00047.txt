Dies ist die Beschreibung der Klasse "CXB_MULTIPLE_STRING - Vergleiche": Die folgenden Übersichten zeigen die Vergleichsoperatoren und die Operanden mit denen ein mehrsprachiger Text verknüpft werden kann. Alle Operatoren geben das Ergebnis des Vergleichs als Wahrheitswert zurück. Gleichheit Größer Kleiner Voranmerkung: Wenn ein mehrsprachige Text mit einem anderen (mehrsprachigen) Text verglichen wird, so wird nur der Text der aktuell eingestellten Landessprache mit dem anderen Text verglichen. Gleichheit ML_STRING = STRING = INTEGER <-> T("abc") = "abc" = 1 T("abc") = "def" = 0 T("abc", "def") = "def" = 0 bzw. 1 ML_STRING = ML_STRING = INTEGER <-> T("abc", "def") = T("abc, "def") = 1 T("abc", "def") = T("abc, "hij") = 0 bzw. 1 T("hij", "def") = T("abc, "def") = 0 bzw. 1 Größer ML_STRING > STRING = INTEGER <-> T("abc") > "abc" = 0 T("def") > "abc" = 1 T("abc", "def") > "abc" = 0 bzw. 1 ML_STRING > ML_STRING = INTEGER <-> T("def", "klm") > T("abc", "def") = 1 T("abc", "hij") > T("def", "klm") = 0 T("abc", "klm") > T("def", "hij") = 0 bzw. 1 Kleiner ML_STRING < STRING = INTEGER <-> T("abc") < "abc" = 0 T("def") < "abc" = 0 T("abc", "def") < "def" = 0 bzw. 1 ML_STRING < ML_STRING = INTEGER <-> T("def", "klm") < T("abc", "def") = 0 T("abc", "hij") < T("def", "klm") = 1 T("abc", "klm") < T("def", "hij") = 0 bzw. 1.
