Objekte dieser Klasse können per CreateTransObject(CX_STRING) oder CreatePersObject(CX_STRING) angelegt werden. Der Inhalt wird per Put zugewiesen. Per String kann auf den enthaltenen STRING zugegriffen werden. 199051 Wird versucht, eine Methode per Call() auf einem STRING/ML_STRING aufgerufen, dann wird dieser String für den Call() implizit in ein temporäres CX_STRING-Objekt umgewandelt, sodass alle hier erwähnten Methoden direkt auf regulären STRINGs und ML_STRINGs (aktive Sprache) aufgerufen werden können. Methoden, die den CX_STRING verändern (Beispiel: FromBytes), haben keine Auswirkung, weil sie lediglich auf einer temporären Kopie des STRINGs arbeiten.
