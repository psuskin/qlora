Dies ist die Beschreibung der Funktionalität der Klasse "CXI_WO_SEED" bezüglich Beschreibung: Nur für Mini Workbench. Diese Hilfsklasse repräsentiert die Widqets. Ein CXI_WO_SEED-Objekt kann mit Drag&Drop auf ein Window gezogen werden. Bei Drop wird das Widqet, das von CXI_WO_SEED-Objekt repräsentiert wird, auf das Window erzeugt. Das Window geht in den Edit-Modus, d. h. alle Widqets können interaktiv verschoben werden. Spezialfall: CXI_WO_SEED mit Name ".Fix" funktioniert wie FixWidgets Window -> Normal-Modus CXI_WO_SEED mit Name ".Loose" funktioniert wie LooseWidgets Window -> Edit-Modus Warnung: Dieses Feature wird ab ClassiX 4.6.0.180614 nicht mehr unterstüzt und das verwenden dieser Klasse erzeugt entsprechende Warnmeldungen im Log.
