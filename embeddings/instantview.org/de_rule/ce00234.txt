Dies ist die Beschreibung der Klasse "CX_TIME - Arithmetik": Die folgenden Übersichten zeigen die mathematischen Operatoren und die Operanden mit denen eine Uhrzeit verknüpft werden kann. Addition CX_TIME + CX_VALUE = CX_TIME -> 9.20 + 5 Minuten = 9.25 Subtraktion CX_TIME - CX_VALUE = CX_TIME -> 9.20 - 5 Minuten = 9.15 Wird eine Uhrzeit von einer anderen abgezogen, so wird die Differenz immer in hundertstel Sekunden ausgegeben. CX_TIME + CX_TIME = CX_VALUE <-> 9.20 + 9.00 = 120000 s/100.
