Dies ist die Beschreibung der Klasse "CX_STRUCTURE - Einfügen": Die Funktionen, die in diesem Bereich zusammengefasst sind, erlauben das Einfügen von neuen Objekten in eine Struktur. Funktionsname Beschreibung InsertFirst Fügt das Element als erstes in die Liste der untergeordneten Elemente ein InsertLast Fügt das Element als letztes in die Liste der untergeordneten Elemente ein InsertBefore Fügt das Element vor einem bestimmten Element ein InsertAfter Fügt das Element nach einem bestimmten Element ein InsertLevelAbove Eine neue Ebene über der Ebene einfügen InsertLevelBelow Eine neue Ebene zwischen der Ebene und untergeordneten Elementen einfügen.
