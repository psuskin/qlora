Dies ist die Beschreibung der Klasse "CX_PRESENTATION_MANAGER": 169434. Dies ist die Beschreibung der Funktionalität der Klasse "CX_PRESENTATION_MANAGER" bezüglich Beschreibung: Der Presentationmanager kümmert sich um die native Darstellung von ClassiX. Hierzu gehört, dass Bitmaps und Cursor ausgetauscht werden können und Positionierungen von Fenstern gespeichert werden. Zudem kann über den Presenationmanager ein anwendungsglobaler Fenstertitelsuffix/-Präfix gesetzt werden. Dies ist die Beschreibung der Funktionalität der Klasse "CX_PRESENTATION_MANAGER" bezüglich Icons / Bitmaps: Grundlage für das Austauschen der Bitmaps ist der BmpIndex, welcher festlegt, welcher Satz an Bitmaps benutzt wird. Die Sätze werden über CX_BITMAP, CX_BITMAP2 etc. definiert. Für Widgets mit Icons wird in der Definition im InstantView-Code der Name eines .bmp oder .png Files angegeben. Diese Filenamen im InstantView-Code können auf unterschiedliche Sets anderer Filenamen gemappt werden. ClassiX kennt dann unterschiedliche Sets von Images für die Oberfläche. Mit dem Presentationmanager wird das aktuell anzuwendende Set ausgewählt. Wichtig: das Mapping bildet einen Filenamen auf anderen, keinen vollständigen Pfad. Diese Feature ist optional! Die Zuordnung (Name im IV-Code) → { IconFile1, IconFIle2, . . . } wird mit Excel beschrieben. Daraus kann das Mapping-File imageMap.json erzeugt werden (siehe prsntmgr.mod und Methode CX_BITMAP_PATH::Save). Dies ist die Beschreibung der Funktionalität der Klasse "CX_PRESENTATION_MANAGER" bezüglich Pfade: Die Regel für Fallback trifft zu, wenn das Mapping-Feature nicht aktiv ist oder wenn da, wo es erwartet wurde, kein Image-File gefunden wurde:. Dies ist die Beschreibung der Funktionalität der Klasse "CX_PRESENTATION_MANAGER" bezüglich Cursor: Die Cursor für laufende Lese/Schreibtransaktion sind über den Presentation-Manager konfigurierbar und können durch eigene Cursor-Dateien ausgetauscht werden (per LoadCursorFromFile). Die Original-Cursor lassen sich per RestoreROCursor/RestoreRWCursor wiederherstellen.
