Dies ist die Beschreibung der Klasse "STRING - Arithmetik": Die folgenden Übersichten zeigen die mathematischen Operatoren und die Operanden mit denen ein Text verknüpft werden kann. Addition STRING + STRING = STRING "Hallo " + "Welt" = "Hallo Welt" STRING + ML_STRING = STRING "Hallo " + T("Welt", "Erde") = "Hallo Welt" bzw. "Hallo Erde" Anmerkung: Die Auswahl, welcher Text von dem mehrsprachigen Text an den Ausgangstext gehangen wird, hängt von der aktuell eingestellten Sprache ab. Diese Operation ist nicht kommutativ. STRING + OBJEKT = STRING "Hallo " + 20€ = "Hallo 20€"(6) Anmerkung: Wird ein Objekt zu einem String addiert, dann wird das Objekt zunächst in einen String umgewandelt (siehe Befehl String) und das Ergebnis dann mit dem ersten String verkettet. Diese Operation ist nicht kommutativ.
