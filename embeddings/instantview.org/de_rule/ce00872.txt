Dies ist die Beschreibung der Klasse "Statische (Objekt ungebundene) Funktion": Eine Funktion, die über SYSTEM Call() aufgerufen wird, ist keinem Objekt einer bestimmten Klasse zugehörig. Sie stellt eine allgemeingültige Berechnungsroutine dar (die Funktion Barcode25Interleaved() errechnet aus einer beliebigen Zeichenkette die Repräsentation für den Barcode 2/5 Interleaved() oder sie stellt Informationen des Systems zur Verfügung (z.B. die Funktion GetLocale()).
