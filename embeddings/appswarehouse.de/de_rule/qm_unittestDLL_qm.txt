Dies ist die Beschreibung des Moduls "qm_unittestDLL_qm" mit dem Namen "Unittest C (plusplus)": Das Unittest C++ Modul stellt eine Testumgebung bereit, die die Durchführung von in C++ geschriebenen Tests in den ClassiX® DLLs ermöglicht, überwacht und anschließend ein Protokoll erstellt. Dieses Modul und weitere Dokumentation über die Tests ist für den Gebrauch in der ClassiX Basisentwicklung vorgesehen. Dies ist die Beschreibung der Funktionalität des Moduls "qm_unittestDLL_qm" mit dem Namen "Unittest C (plusplus)" bezüglich Funktionalität: Es werden alle C++-Tests aufgelistet. Über den Knopf können die Tests gestartet werden: Alle, wenn keine so genannte Suite ausgewählt ist oder *, oder nur die ausgewählte Suite. Ein Suite bezeichnet eine Zusammenfassung von mehreren Tests. Auf C++-Ebene entspricht eine DLL einer Suite Im mittleren Bereich des Fensters wird der Fortschritt der durchgeführten Tests angezeigt. Die Zahl rechts neben dem Fortschrittsbalken zählt die Tests. Nach Ausführung der Tests ist der Fortschrittsbalken entweder grün oder rot eingefärbt: grün bedeutet, dass alle Tests fehlerfrei durchliefen, rot bedeutet, dass mindestens ein Test fehlschlug. Im unteren Teil des Fensters werden in einer Liste alle Fehler gesammelt, die während der Testausführung auftraten.
