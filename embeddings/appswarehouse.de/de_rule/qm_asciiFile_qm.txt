Dies ist die Beschreibung des Moduls "qm_asciiFile_qm" mit dem Namen "Datei Objekt (CX_ASCII_FILE)": Dieser Test überprüft den korrekten Umgang mit ASCII-Dateien. Hierzu wird eine Datei erstellt und mit Beispieldaten gefüllt. Danach wird die Datei wieder ausgelesen und der gelesene Inhalt dargestellt. Der Testdurchlauf wird nach dem Aufruf automatisch gestartet und lässt sich über das Menü ‚Start' beliebig oft wiederholen. // 1. Create vector with test data // 2. Write test data into a file with fix record format (tstfix.dat) // 2. Build index file (tstfix.idx) // 3. Write test data into file with variable record format (tstvar.dat) // 4. Build index file (tstvar.idx) // 5. Read file in fix format (with translation table) and compare with test data vector // 6. Seek some records // 7. Read file in variable format (with translation table) and compare with test data vector // 8. Seek some records.
