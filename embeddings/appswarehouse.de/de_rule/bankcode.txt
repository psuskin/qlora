Dies ist die Beschreibung des Moduls "bankcode" mit dem Namen "Bankleitzahlen": ClassiX® verwaltet ein Register von Bank Codes bzw. Bankleitzahlen, um Sie vor Fehleingaben zu schützen. Wenn Sie irgendwo im Programm ein Bank Code mit RETURN bestätigen, dann wird in der Bank Code/BLZ-Liste nach einem entsprechenden Eintrag gesucht. Sollten mehrere Einträge zu dem eingegebenen Bank Code vorhanden sein, so können Sie die richtige Bank aus einer Liste auswählen.Es kommt sehr häufig vor, dass Sie unter einem Bank Code mehrere Einträge vorfinden werden, dieses gilt auch für den SWIFT Code. Diese Codes sind nicht eindeutig. Intern werden die einzelnen Einträge jedoch über eine - eindeutige - laufende Nummer des Datensatzes unterschieden.Dieses Register kann aus Dateien, die von Zentralbanken zur Verfügung gestellt werden, geladen werden (s.u.). Sie können dieses auch mehrfach tun, um Ihr Register zu aktualisieren. Bei der Aktualisierung wird der Datensatz basierend auf seiner laufenden Nummer überprüft. Da nur diese laufende Nummer des Datensatzes eine eindeutige Zuordnung erlaubt, ist es für den Anwender nicht möglich dieses Feld zu verändern. Dies ist die Beschreibung der Funktionalität des Moduls "bankcode" mit dem Namen "Bankleitzahlen" bezüglich Eingabefenster: Dieses Fenster dient der Pflege von Bank Code/BLZ. Sie können einen Bank Code auswählen, in dem Sie im Feld Nummer die gewünschte Nummer eingeben und mit RETURN bestätigen. Wenn Sie das Listenfenster aufrufen wollen, dann können Sie den Menüpunkt "Suchen" auswählen. Dies ist die Beschreibung der Funktionalität des Moduls "bankcode" mit dem Namen "Bankleitzahlen" bezüglich Listenfenster: Dient der Auflistung der Bank Codes für die Bankfilialen für den nationalen Zahlungsverkehr. Dies ist die Beschreibung der Funktionalität des Moduls "bankcode" mit dem Namen "Bankleitzahlen" bezüglich Selektionsfenster: Dieses Fenster dient der Selektion einer Bankleitzahl.
