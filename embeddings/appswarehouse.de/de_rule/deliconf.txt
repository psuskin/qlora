Dies ist die Beschreibung des Moduls "deliconf" mit dem Namen "Lieferterminbestätigung": Dieses Fenster dient der Schnellerfassung von Lieferterminbestätigungen von Bestellungen. Dies ist die Beschreibung der Funktionalität des Moduls "deliconf" mit dem Namen "Lieferterminbestätigung" bezüglich Eingabefenster: Über ‚Bestellung’ öffnet sich das Fenster ‚Bestellpositionen (alle)’. Dieses Fenster ermöglicht Ihnen eine Selektion der Bestellungen mit ‚Bearbeiten - Filter’ oder ‚-Auftrag’. Es werden anschließend alle, der Selektion entsprechenden, Bestellpositionen angezeigt. In dem Sie aus den Positionen die bestätigte Bestellposition anklicken und mit ‚OK’ bestätigen, wird diese in das Fenster ‚Lieferterminbestätigung’ übernommen. Anschließend können Sie im Feld ‚AB-Lieferant’ AB-Nummer des Lieferanten, die Bestellnummer und den bestätigten Termin eingeben. Diese Daten werden dann automatisch in die Bestellposition übernommen und der Status der Position auf „Bestellposition bestätigt“ gesetzt. Den Liefertermin kann man auch über einen Doppelklick in die Bestellposition editieren. Werden mehrere Positionen bestätigt, können Sie die auch mehrere Bestellpositionen im Fenster ‚Bestellpositionen (alle)’ markieren und mit der gedrückt gelassenen, linken Maustaste in das Fenster ‚Lieferterminbestätigung’ ziehen. Anschließend können Sie wieder die AB-Nummer und den bestätigten Termin eingeben. Diese Daten werden in den Positionen gespeichert und der Status automatisch auf „Bestellung bestätigt“ gesetzt. Mit dem ‚OK’ speichern Sie die Daten ab. Bitte geben Sie die Auftragsbestätigungsnummer des Lieferanten ein, wählen dann den oder die Bestellpositionen aus. Wenn der Lieferant Ihnen die gewünschten Liefertermine bestätigt, können Sie einfach den OK-Knopf betätigen. Sollte der Liefertermin abweichen, so können Sie entweder im Feld Liefertermin den bestätigten Termin erfassen, oder Sie können in der Liste den Termin einer Position mit Doppelklick editieren. Sie müssen den OK-Knopf betätigen, um die Änderungen zu speichern. Das Fenster wird anschließend automatisch geleert. Sollten Sie schon eine Auftragsbestätigungsnummer bestätigt haben und Sie wollen nur noch den Liefertermin ändern, so lassen Sie einfach das Feld AB-Lieferant frei, ändern den oder die Positionen per Doppelklick. Die gewünschten Änderungen werden sofort in die Datenbank geschrieben. Sie können eine Bestellung auswählen, in dem Sie im Feld Bestellung gewünschte Nummer eingeben und mit RETURN bestätigen. Wenn Sie das Bestellungs-Selektionsfenster aufrufen wollen, dann können Sie den Knopf / Menüpunkt "Bestellung" auswählen.
