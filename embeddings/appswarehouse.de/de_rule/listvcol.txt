Dies ist die Beschreibung des Moduls "listvcol" mit dem Namen "Gespeicherte Listen": Viele der Listen/Auswertungen im AppsWarehouse® von ClassiX® können über die Symbole aus einem Zwischenspeicher aufgerufen, gelöscht und gespeichert werden. Diese Funktionalität erlaubt es zum Einen eine einmal aufgerufene Liste zwischenzuspeichern und zu einem anderen Zeitpunkt wieder aufzurufen. Zum Anderen kann ein Benutzer eine Liste, für die dieser Benutzer ein Zugriffsrecht besitzt, einem anderen Benutzer - der nicht dieses Zugriffsrecht hat - zur Ansicht überlassen. Eine so übergebene, gespeicherte Liste kann in der Nutzung eingeschränkt sein (siehe dazu die Dokumentation der jeweiligen Auswertung). Eine Liste wird jeweils pro Benutzer gespeichert, ein Benutzer kann aber auch mehrere Versionen der gleichen Liste speichern. Dies ist die Beschreibung der Funktionalität des Moduls "listvcol" mit dem Namen "Gespeicherte Listen" bezüglich Listenfenster: Dieses Fenster zeigt alle die vom angemeldeten Benutzer gespeicherten Listen an. Diese werden mit Kategorie, Listenbezeichnung, Listenidentifikation (z.B. Benutzerkürzel), Anlagedatum und Anzahl an Listeneinträgen ausgegeben. Die Kategorie kann benutzt werden, damit ein Benutzer mehrere verschiedene Inhalte der selben Liste speichern kann. Über die rechte Maustaste kann man eine selektierte Liste einem anderen Benutzer übergeben. Dieser Benutzer kann die Liste auch dann einsehen, wenn er eigentlich keine Berechtigung zum Erstellen der Liste hat. Außerdem kann per rechter Maustaste der Text der gespeicherten Liste editiert werden. Dies ist die Beschreibung der Funktionalität des Moduls "listvcol" mit dem Namen "Gespeicherte Listen" bezüglich Selektionsfenster: Dieses Fenster dient der Selektion der gespeicherten Listen. Es werden nur die für die aktuelle Liste gespeicherten Listeninhalte aller Benutzer angezeigt. Dies ist die Beschreibung der Funktionalität des Moduls "listvcol" mit dem Namen "Gespeicherte Listen" bezüglich Selektion Version: Dieses Fenster dient der Selektion einer Version einer mehrfach gespeicherten Liste. Dieses Fenster öffnet sich dann, wenn man eine Liste speichern will, die bereits mehr als einmal gespeichert worden ist - eine Liste also, die eventuell zu verschiedenen Zeitpunkten gespeichert wurde oder mit verschiedenen Parametern aufgerufen und dann jeweils gespeichert wurde.
