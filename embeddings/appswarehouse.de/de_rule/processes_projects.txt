Diese Grobplanung ist oft vom Auftraggeber vorgegeben, z.B. wenn das Unternehmen selbst als Subunternehmer in einem größeren Projekt auftritt. Als Tool für die Projektplanung ist Microsoft Project weit verbreitet, weshalb die Planung oft in diesem Format schon vorliegt. Um in ClassiX® mit der Planung der eigenen Ressourcen beginnen zu können, muss diese Grobplanung entsprechend übernommen werden. Dazu stellt ClassiX® eine Schnittstelle zu Microsoft Project zur Verfügung. Diese Schnittstelle ist so leistungsfähig, dass auch spätere Änderungen an der Grobplanung seitens des Auftraggebers bei schon verfeinerter Planung in ClassiX® noch übernommen werden können. Falls die Grobplanung wieder im MS Project Format benötigt wird, stellt ClassiX® auch noch eine Exportfunktion bereit. .
