Dies ist die Beschreibung der Funktionalität des Moduls "viewtest_web" mit dem Namen "Widget Web Test" bezüglich Einleitung: In diesem Modul befinden sich alle Widgets (jedes einmal). Alle Daten werden entsprechend ihres Datentyps in einen Slot in einem CX_PERSON Object gespeichert. Dies ist die Beschreibung der Funktionalität des Moduls "viewtest_web" mit dem Namen "Widget Web Test" bezüglich Editieren: Ein Objekt kann gespeichert werden indem man auf den OK-Button oder auf den Button mit der Diskette im Toolbar klickt. Jedes weitere Objekt das erzeugt werden soll muss mit dem 'Neu'-Button erzeugt und mit dem OK-Button gespeichert werden. Im zweiten Reiter (Hobbys) können nach anlegen des Objekts ListView Einträge dem Objekt hinzugefügt werden. Dies ist die Beschreibung der Funktionalität des Moduls "viewtest_web" mit dem Namen "Widget Web Test" bezüglich Suchen: Gespeicherte Objekte können durch den im Eingabefeld String eingegebenen Wert in der Liste identifiziert und wieder geladen werden.
