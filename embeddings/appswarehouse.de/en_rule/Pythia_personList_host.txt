This is the description of the functionality of the module "Pythia_personList_host" with the name "List of persons Host" regarding List window: This window is used to list the persons. On the right of the window is a list of the roles and communication objects of the currently selected person. Double-clicking on a role takes you directly to its input window. Furthermore, the context menu can be called up with a right click to create new roles or to display the validity. Communication objects can be edited with a double click.
