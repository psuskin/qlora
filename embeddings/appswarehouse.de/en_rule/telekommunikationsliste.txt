This is the description of the module "telekommunikationsliste" with the name "Telecommunications list": The telecommunications list is used to display all telecommunications information on several objects, e.g. the employees of a company. This includes the telephone number, mobile phone and fax number as well as e-mail address and Internet address. This is the description of the functionality of the module "telekommunikationsliste" with the name "Telecommunications list" regarding Telecommunication list normal in the ClassiX® application: The telecommunication list in ClassiX® itself can be found e.g. in the selection mask for a company under the telephone button. For further information on the telecommunication list in ClassiX® please refer to the module documentation.
