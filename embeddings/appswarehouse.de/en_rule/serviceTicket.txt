This is the description of the module "serviceTicket" with the name "Service tickets (support staff)": Description Functionality Technical documentation. This is the description of the functionality of the module "serviceTicket" with the name "Service tickets (support staff)" regarding Description: This module is used to manage service tickets by the internal support. It offers the possibility to process tickets for errors, feature requests and maintenance requests. Additional information can be provided by attaching notes and documents to the ticket. Communication with the customer is also handled via notes. This is the description of the functionality of the module "serviceTicket" with the name "Service tickets (support staff)" regarding List window: This window shows the overview of all tickets. It offers filter functions for various features of the ticket as well as a full text search in the subject, description and notes of the tickets. From here, new tickets can be entered and existing ones processed. Buttons. This is the description of the functionality of the module "serviceTicket" with the name "Service tickets (support staff)" regarding Input window: Tickets can be edited or entered in this window. Buttons. This is the description of the functionality of the module "serviceTicket" with the name "Service tickets (support staff)" regarding Add a note: A new ink note can be entered in this window. At the same time, the status of the ticket can be changed, an editor assigned and the text of the note sent to the reporter as an e-mail. Fields Buttons.
