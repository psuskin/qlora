This is the description of the module "consult" with the name "Consultant": The colloquially established term "consultant" combines the German professional titles "Berater" and "Planer", which is why it was retained in the German language version. This is the description of the functionality of the module "consult" with the name "Consultant" regarding Input window: This module is used to maintain the consultant data. You can select a consultant by entering the consultant number in the Number field and confirming with RETURN. If you want to call up the consultant list, you can either double-click on the "Number" field or select the "Search" menu item. Fields marked with "italics" represent data directly from the company. If something is changed in one of these fields, the change is applied to all roles and the company itself. All this data can also be maintained directly in the company input window. This is the description of the functionality of the module "consult" with the name "Consultant" regarding List window: This window serves to list the consultants. This is the description of the functionality of the module "consult" with the name "Consultant" regarding Selection window: This window is used to select a consultant. This is the description of the functionality of the module "consult" with the name "Consultant" regarding Option window: All settings for the search can be made here. The window can only be closed with the "OK" button.
