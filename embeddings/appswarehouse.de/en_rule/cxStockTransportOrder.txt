This is the description of the module "cxStockTransportOrder" with the name "Warehouse Transport order": Module for maintaining transfer orders in a warehouse. This is the description of the functionality of the module "cxStockTransportOrder" with the name "Warehouse Transport order" regarding Input window: This window is used for the maintenance of storage. You can select a putaway by entering the desired number in the Number field and confirm with RETURN. If you want to call up the list window, you can select the menu item "Search". If you want to enter a new putaway, press the OK button (number, date and employee are preset). Then you can add the putaway items. After that you can enter stock items that have been entered in the goods receipt into the warehouse (GR button) or enter other putaways (e.g. put back surplus goods ) (Insert button). With the button 'Insert' you can add items to the document that are not in the goods receipt list. The part master opens from which you can select and post further parts. With the button 'Delete' you can remove items from the document and cancel your booking. This is the description of the functionality of the module "cxStockTransportOrder" with the name "Warehouse Transport order" regarding Position window: The item number is assigned automatically. For put-away that was not triggered by a goods receipt, the cost object can be assigned via menu or by right-clicking. This is the description of the functionality of the module "cxStockTransportOrder" with the name "Warehouse Transport order" regarding List window: Serves to list the storage locations. This is the description of the functionality of the module "cxStockTransportOrder" with the name "Warehouse Transport order" regarding Selection window: This window is used to select a putaway.
