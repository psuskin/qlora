This is the description of the module "cxTool" with the name "Tool (module)": Tools are defined as part roles in the CyberEnterprise® enterprise model. This means that tools can also have other part roles, such as ordering and production roles, or be managed as assets in the asset master. They can also be kept in stock. The part assigned to a tool is called a resource and is also managed separately to the part master. In contrast, in the CyberEnterprise® enterprise model, machines and vehicles(cars and trucks) are considered as special manifestations of parts themselves. This is the description of the functionality of the module "cxTool" with the name "Tool (module)" regarding Input window: This window is used to maintain tools. You can select a tool by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". This is the description of the functionality of the module "cxTool" with the name "Tool (module)" regarding List window: Serves to list the tools. This is the description of the functionality of the module "cxTool" with the name "Tool (module)" regarding Selection window: This window is used to select a tool. This is the description of the functionality of the module "cxTool" with the name "Tool (module)" regarding Option window: All settings for the search can be made here. The window can only be closed with the "OK" button.
