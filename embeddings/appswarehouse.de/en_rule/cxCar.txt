This is the description of the module "cxCar" with the name "CAR": In the CyberEnterprise® Enterprise Model, vehicles are treated as a special form of parts, i.e. according to the terminology of object orientation, vehicles are also said to be derived from parts and vehicles are inherited from parts. Another specialisation of vehicles are cars and trucks. Vehicles are maintained in the CyberEnterprise® enterprise model as part of a company's vehicle fleet. E.g. an assignment to a car is stored in the route plan of a delivery. This is the description of the functionality of the module "cxCar" with the name "CAR" regarding Input window: This window is used for car maintenance. You can select a car by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". This is the description of the functionality of the module "cxCar" with the name "CAR" regarding List window: Serves to list the cars in the fleet.
