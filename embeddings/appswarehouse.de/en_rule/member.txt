This is the description of the module "member" with the name "Member - Dynamic data fields": This application allows you to list existing members and create new members.A member refers to a data field of the type ENUMCHAR, ENUMSHORT or ENUMINT of a C++ class and defines the enumeration table for this data field.One can thus specify the names for the individual values for the C++ class members externally in the form of such enumeration or enumeration tables. This is the description of the functionality of the module "member" with the name "Member - Dynamic data fields" regarding List window: In this list you can see all members available in the system. This is the description of the functionality of the module "member" with the name "Member - Dynamic data fields" regarding Selection window: This window is used to select a member.
