This is the description of the functionality of the module "taxauth" with the name "Tax Office Modules" regarding Description: It is then necessary to maintain data on tax offices in order to be able to exchange data with a tax office, especially electronically (e.g. see advance return for VAT). For the Federal Republic of Germany, this data can be obtained via the ELSTER interface. This is an online query about ELSTER. The data listed in this window can be used as a template for data maintenance. This is the description of the functionality of the module "taxauth" with the name "Tax Office Modules" regarding List window: This window lists the tax offices stored in the database. This is the description of the functionality of the module "taxauth" with the name "Tax Office Modules" regarding Selection window: This window is used to select a tax office.
