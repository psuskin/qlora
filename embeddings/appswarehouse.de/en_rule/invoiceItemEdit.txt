This is the description of the module "invoiceItemEdit" with the name "Accounting position": Order items can be invoiced via invoice items. Invoice items are always part of an invoice. This is the description of the functionality of the module "invoiceItemEdit" with the name "Accounting position" regarding Functionality: This window is for maintaining invoice items. You can select an invoice item by entering the desired number in the Number field and confirming with RETURN. If you want to open the list window, you can select the menu item "Search".
