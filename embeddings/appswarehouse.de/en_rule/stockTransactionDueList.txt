This is the description of the module "stockTransactionDueList" with the name "Stock movements to be carried out": This module can be used to list all types of stock movements still to be processed (putaways, stock removals and stock transfers) at a glance. In particular, this module is used in the context of creating transfer orders. This is the description of the functionality of the module "stockTransactionDueList" with the name "Stock movements to be carried out" regarding List window: This window is used to list the stock movements still to be processed. This is the description of the functionality of the module "stockTransactionDueList" with the name "Stock movements to be carried out" regarding Option window: All settings for the selection of stock movements can be made here.
