This is the description of the module "processes_docind_main" with the name "List of documents": A document directory is the interface between business objects, such as parts or part roles, and documents. A document can be a CAD drawing, a PDF or any other file. It does not matter whether the file is stored in the database or in a server directory. Furthermore, a document directory can also be assigned to a document. On the left: List of documents Documents as QS report Procurement documents.
