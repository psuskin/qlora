This is the description of the functionality of the module "objectChangeLogList" with the name "Evaluation of change history" regarding Description: This program is for evaluating the change history created by this machine. The data for a certain period of time can be read in and displayed for selected objects. First, the period for which the changes are to be displayed should be selected and the data read in using the "Read in log files" button. This can take some time depending on the selected period. Then the objects to which changes are to be displayed can be added by drag & drop or by clicking on the "Read object" button. The changes to the selected objects are displayed by clicking on the "Search" button. Once the data has been read in, further or other objects can be selected and their change history displayed.
