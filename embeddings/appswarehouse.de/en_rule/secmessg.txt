This is the description of the module "secmessg" with the name "Access rights for messages": The access groups are used to manage user rights (see also Access Rights). User actions, such as pressing a button, can be used to send messages that cause a mask to be called up or data to be changed. To block this action for some users, a message block can be created via this module. This is the description of the functionality of the module "secmessg" with the name "Access rights for messages" regarding Input window: With the insert and remove button, a new message can be added to or removed from the definition. Each entry of the list can be edited via Oboxedit, i.e. a double click on the list element.
