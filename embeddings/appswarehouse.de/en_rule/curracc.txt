This is the description of the module "curracc" with the name "Current account": Current or cash accounts are managed by banks and the "custodian" partner role assigned to them. Their main function in ClassiX® is to maintain the connection to the money accounts in the financial accounting. Bank statements provided electronically by banks are assigned to the accounts maintained here after being read into ClassiX®(bank statements) and can also be viewed here. This is the description of the functionality of the module "curracc" with the name "Current account" regarding Input window: This window is used to maintain current accounts. It is divided into the three partial flaps. This is the description of the functionality of the module "curracc" with the name "Current account" regarding Excerpts: If bank statements have been imported for the selected account, they are displayed in this window separately by posting year. This is the description of the functionality of the module "curracc" with the name "Current account" regarding List window: Serves to list the current accounts.
