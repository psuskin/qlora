This is the description of the module "cxTruck" with the name "TRUCK": In the CyberEnterprise® Enterprise Model, vehicles are treated as a special form of parts, i.e. according to the terminology of object orientation, vehicles are also said to be derived from parts and vehicles are inherited from parts. Another specialisation of vehicles are trucks and cars. Vehicles are maintained in the CyberEnterprise® enterprise model as part of a company's fleet. E.g. an assignment to a truck is stored in the route plan of a delivery. This is the description of the functionality of the module "cxTruck" with the name "TRUCK" regarding Input window: This window is used for truck maintenance. You can select a truck by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". This is the description of the functionality of the module "cxTruck" with the name "TRUCK" regarding List window: Used to list the trucks of the fleet.
