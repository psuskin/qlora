pcalcgrp. Input window: This window is used to maintain pricing groups. You can select a pricing group by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". List window: Used to list the pricing groups. Selection window: This window is used to select a pricing group. Subgroup window: This window allows you to insert a pricing group subgroup. Module name: pcalcgrp.mod. Classes: CX_STRUCTURE.
