extdispo: External dispositions can be used to manually post documents about planned receipts/issues in the logistics account. This method can be used to compare the dispositions of a second system and for testing(see quality assurance). Input window: Especially when comparing external dispositions, you should have the disposition positions created automatically via the menu item "Load external data". The same step can also be done manually. By saving a new disposition (New) an "External disposition" is created. Afterwards you can insert/delete new positions in the list by right mouse click. The type of item determines whether it is a receipt or issue document. List window: This window is used to list external schedules. Selection window: This window is used to select external dispositions. Module name: extdispo.mod. Classes: CX_EXTERNAL_DISPOSITION.
