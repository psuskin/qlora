purchaseItem: An order item manages all information relevant for purchasing. Input window: In the input window, all data for order items are maintained. To select an order article directly, enter the desired number in the Number field and confirm with RETURN. To search for order items in the list window, select the menu item "Search". To create a new order item, please first create a new item in the parts master. Via the menu item 'Edit - Purchasing data' the window 'Order items from .' opens. List window: Serves to list the articles ordered. In this mask all available order items are displayed. You can search for individual entries by number, description or purchasing area or sort the complete list. With a double click on the desired entry the mask 'Bestellartikel von .' (order item from .) is opened where you can now edit all data entered there. With 'OK' you can save your changes. Selection window: This window is used to select an order item. Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Technical documentation: Has changed:. Module name: pitem.mod.
