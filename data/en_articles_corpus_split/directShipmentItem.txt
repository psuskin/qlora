directShipmentItem: Positions of the direct delivery notes. Input window: This window is used to maintain direct delivery items. The direct delivery should be matched with the orders as immediately as possible. To do this, you must press the order button and transfer the desired order item(s) to the item either by pressing a button or by dragging and dropping. If you have selected several order items, a direct delivery item is automatically created for each order item. Direct deliveries that cannot be immediately assigned to an order item should receive a corresponding comment in the Description field. Another employee can then make this assignment more easily. These direct deliveries can be found with the function "Unresolved direct delivery" in the warehouse menu. List window: Used to list the direct delivery items.
