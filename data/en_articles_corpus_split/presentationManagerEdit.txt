presentationManagerEdit: This window provides various setting options for customising the icons and cursors. Furthermore, the description texts for the icons are maintained here, which are stored in the system file imageMap.json. These are then integrated in the browser as alternative texts. Input window: This window is used to maintain the mapping entries. The fields can also be maintained later in the list view. The environment variables CX_BITMAP and CX_BITMAP2 define the paths to the respective image version in the calling batch file. List window: This window is used to list all image entries stored in the system file imageMap.json. Direct editing is done by double-clicking on the respective list entry. Module name: prsntmgr.mod. Classes: CX_"Class.
