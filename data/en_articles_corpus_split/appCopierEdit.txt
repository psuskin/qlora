appCopierEdit: This app allows you to copy a new app from an existing app / module. It is thus part of the apps with which you can create new apps: Derive app Copy app Generate app. Input window: This window is used to enter the parameters for copying an app. Module name: appCopier.app. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the messages it receives.
