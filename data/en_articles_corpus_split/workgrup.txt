workgrup: This module is used to maintain working groups. Through working groups, individual employees are organised into work departments. Input window: This window is used for maintaining workgroups. You can select a working group by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". Enter a name for the group. With 'Group leader' the list of employees opens. This allows you to select the appropriate employee directly with a double click or with 'OK'. All current workstations are stored in the 'Workstations' field and you only need to select the appropriate one. In the field 'Standard costs' you can enter a value of your choice. Via the menu item 'Edit - Group members - Insert' the list of employees opens again, from which you can select the individual employees of the group. Press 'OK' to save your entries. List window: Serves to list the working groups. Selection window: This window is used to select a working group. Module name: workgrup.mod. Classes: CX_WORK_GROUP.
