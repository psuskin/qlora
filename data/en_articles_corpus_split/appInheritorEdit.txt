appInheritorEdit: With this app, a derivative (inheritance) of an app to be selected beforehand can be generated. In the newly generated, derived (inherited) app, changes and/or extensions to the derived app can then be made.It is thus part of the apps with which one can create new apps:Derive appCopy appGenerate app. Input window: This window is used to enter the parameters for the generator of a derivative. Fields. Module name: appInheritorEdit.app. Classes: CX_MODULE CX_WIDGET. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages.
