operator. Description: An operator is a relationship role of a company or person between itself and your company. Input window: This module is used to maintain operator data. Fields marked with "italics" represent data directly from the company. If something is changed in one of these fields, the change is adopted for all roles and the company itself. All these data can also be edited directly in the company input window. List window: This window is used to list the operators. Selection window: This window is used to select an operator. Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Module name: operator.mod. Classes: CX_OPERATOR. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages. Module name: operaedt.mod.
