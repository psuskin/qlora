dtazvedt: The paperless data medium exchange with banks for foreign payments (Germany) is carried out using the DTAZV file format. This module is suitable for reading in a file of this format and displaying the data in a prepared form. Analyse DTAZV file: The purpose of reading in a file in DTAZV format is to display the contents of the file in an edited form and to check the error-free structure of the file.
