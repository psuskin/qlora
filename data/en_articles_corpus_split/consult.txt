consult: The colloquially established term "consultant" combines the German professional titles "Berater" and "Planer", which is why it was retained in the German language version. Input window: This module is used to maintain the consultant data. You can select a consultant by entering the consultant number in the Number field and confirming with RETURN. If you want to call up the consultant list, you can either double-click on the "Number" field or select the "Search" menu item. Fields marked with "italics" represent data directly from the company. If something is changed in one of these fields, the change is applied to all roles and the company itself. All this data can also be maintained directly in the company input window. List window: This window serves to list the consultants. Selection window: This window is used to select a consultant. Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Module name: consuedt.mod.
