taxauth. Description: It is then necessary to maintain data on tax offices in order to be able to exchange data with a tax office, especially electronically (e.g. see advance return for VAT). For the Federal Republic of Germany, this data can be obtained via the ELSTER interface. Input window: This module is used for the maintenance of tax office data. List of German tax offices from ELSTER interface: All German tax offices are listed in this window. This is an online query about ELSTER. The data listed in this window can be used as a template for data maintenance. List window: This window lists the tax offices stored in the database. Selection window: This window is used to select a tax office. Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Module name: taxauth.mod. Classes: CX_TAX_AUTHORITY.
