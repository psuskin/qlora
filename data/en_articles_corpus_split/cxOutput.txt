cxOutput. Input window: This window is used to maintain output objects. You can select an output object by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". List window: Used to list the output objects. Selection window: This window is used to select an output object. Address selection window: This window is used to select an address for the output object. Output queue: Display the elements of the output queue. Depending on the selected status, you can see the created, sent, received and printed elements of the queue here. In addition, a time period can be specified. The output queue can be processed from the current database or via a web service. Files received via the web service are stored in a directory defined by the environment variable "CX_QUEUESRV_TEMP" or "TEMP". The addresses of the individual instances can be defined via web services on the client. The function of the web service is defined in the remarks field of the web service. Module name: output.mod. Classes: CX_OUTPUT. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages.
