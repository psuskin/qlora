loggings: Selection window for transactions posted in one or more status monitors. List window: You can select one or more transactions and transfer them with the OK button. If you select a transaction by double-clicking, the corresponding input window is called up. Module name: loggings.mod. Classes: CX_TRANSACTION. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages. Module name: loggisel.mod. Classes: CX_TRANSACTION.
