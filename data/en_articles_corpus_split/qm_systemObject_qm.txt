qm_systemObject_qm: The test for SystemObject tests all functions in SystemObject. After starting the module, a long log window is displayed. The normal test can be initiated with the menu item 'Start'. The results of the individual tests appear in the log window. Via the menu item 'Advanced' another window can be opened, which contains all tests that require user interaction. In the middle of the window you can see various combo boxes that have been filled with special functions. The buttons for the remaining tests are explained on the mask. Module name: reload_qm.mod.
