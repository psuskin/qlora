publisher. Description: A partner function is defined as a publisher for companies or persons who make publications. In general, this applies to publishers, but also to other institutions or persons who make publications. Publications are understood to be print or electronic media. Input window: This module is used to maintain the data of a publisher. You can select a publisher by entering the publisher number in the Number field and confirming with RETURN. Fields marked with "italic" writing represent data directly from the company/person. If something is changed in one of these fields, the change is applied to all roles and the company/person itself. All these data can also be maintained directly in the company input window. All menu items are still locked if a new publisher is entered. List window: This module is used to list editors. Selection window: This window is used to select editors. Module name: publishr.mod. Classes: CX_PUBLISHER. Module name: publiedt.mod. Classes: CX_PUBLISHER. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages. Module name: press.mod. Classes: CX_PUBLISHER.
