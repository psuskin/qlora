personNamesList: This window is used to list the names and titles of persons. For this purpose, the titles, first names, surnames and intermediate names of all persons are collected and displayed in a list of names. The aim of this application is to be able to check names for their correct spelling. Toolbar Symbol Description Rebuild the list of names Search fields Field Description List of names Filter - : : Personal name/title Enter a (partial) name or title. The result is a list of names/titles. If you select an entry in the list, the persons who carry this name or title will be listed below it. : Names Listing of search results : People Listing of the persons selected by name Buttons button Description Action - back The window is closed.
