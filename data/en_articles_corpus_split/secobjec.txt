secobjec: With this module, access rights can be assigned for individual objects. Objects with the same rights (no rights, read only, write only, read and write only, etc.) can be grouped together for this purpose. (see also access rights). Input window: In this window, access rights can be assigned to individual objects. Objects are inserted into this access right object by drag&drop from another list. List window: This window lists the object access rights. Selection window: This window is used to select an object access right. Module name: secobjec.mod. Classes: CX_OBJECT_SECURITY.
