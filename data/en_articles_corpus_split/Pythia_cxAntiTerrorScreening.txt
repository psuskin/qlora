Pythia_cxAntiTerrorScreening: Each import of the EU, US, UK, CH and UN sanctions lists, each individual search, each mass comparison with data from an ERP system or with Outlook contacts (direct comparison with Outlook contacts not available in cloud version) is saved as an anti-terror screening log with the corresponding results. Each log is archived with user, date and time, the search criteria and the results found and can be viewed at any time. Input window: This window displays logs of search, import and matching activities in the Pythia database. The data serves as a log and cannot be changed. List window: This window is used to list the logs of search, import and matching activities in the Pythia database. The data serves as a log and cannot be changed. Selection window: This window is used for the selection . Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Module name: antiterr.mod. Classes: CX_"Class. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages.
