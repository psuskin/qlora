prodserv: The in-house service is a partial roll and contains the relevant production data. This module is used to maintain own services. Input window: In the input window, all data for own services are maintained. To select an own service directly, enter the desired number in the Number field and confirm with RETURN. To search for own services in the list window, select the menu item "Search". This window is divided into the different flaps. List window: Serves to list the own service. In this mask all existing own services are displayed. You can sort individual entries by number, search term, cost element or the complete list. By double-clicking on the desired entry, the mask 'Own service from .' opens, in which you can now revise all data entered there. With 'OK' you can save your changes. Selection window: This window is used to select an own service. Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Module name: prodsedt.mod. Classes: CX_PRODUCT_SERVICE.
