acntstat. Input window: This window is for maintaining account sheets. You can select an account sheet by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". This menu item enables you to output and print your account sheets. From the account sheets, you can double-click on the flows to branch directly to the posting journal. List window: Used to list the account sheets. Selection window: This window is used to select an account sheet. Module name: acntstat.mod. Classes: CX_MONITOR. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages. Module name: fiacstat.mod. Classes: CX_MONITOR.
