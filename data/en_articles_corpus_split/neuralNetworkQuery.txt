neuralNetworkQuery: This app can be used to call up and query networks stored in the system. Query window: After selecting a neural network stored in the system, the possible input and output fields are displayed. If the input fields are filled in, the neural network can be queried and its output is displayed in the output fields. Module name: neuralNetworkQuery.app. Classes: CX_NEURAL_NETWORK. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages.
