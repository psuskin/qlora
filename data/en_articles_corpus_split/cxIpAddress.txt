cxIpAddress: This module is used to maintain IP addresses. IP addresses can be assigned to network objects (e.g. computers) or also restrict incoming and outgoing access (e.g. API keys). In addition, the validity period of the IP address can be defined. Input window: This window is used to maintain IP addresses. The window is divided into two tabs. The most important is the first tab "IP Address". It contains the option of defining the IP address as a text entry for the selected object. The second tab "Characteristics" offers the possibility to add the characteristics (attributes) defined in ClassiXÂ® to an IP address and to assign corresponding values. List window: This window is used to list the IP addresses. Double-clicking on an IP address takes you directly to its input window. Selection window: This window is used to select an IP address. Options window: All settings for the search can be made here. Module name: cxIpAddress.mod. Classes: CX_IP_ADDRESS. Module name: ipAddressEdit.app. Classes: CX_IP_ADDRESS. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages.
