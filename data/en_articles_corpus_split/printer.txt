printer: This module enables you to manage printers for this machine. These can be accessed via the Windows driver or directly via an IP or the printer queue. There are imports of the existing Windows printers and printers from a server via a web service interface. Input window: This window is used for printer maintenance. List window: This window is used to list the printers. Selection window: This window is used to select a printer. Option window: All settings for the search can be made here. Module name: printer.mod. Classes: CX_PRINTER. Module name: printedt.mod. Classes: CX_PRINTER. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages. Module name: printsel.mod. Classes: CX_PRINTER.
