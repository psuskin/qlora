saleserv. Input window: This window is used to maintain services. You can select a service by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". You can add additional data and customer prices on the cards. Additional data can also be used here to assign and maintain characteristics. List window: Serves to list the service objects. Selection window: This window is used to select a service. Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Once set, options can be saved as the default option for the logged in user. For this purpose, there is a context menu item on the "Preferences" tab in the "User Preferences" list which can be used to save the current options. Module name: saleserv.mod. Classes: CX_SALES_SERVICE.
