itemDispositionLoggingsSelect: Selection window for sales order, stock order and advance requirement items that have a completely exploded and technically released bill of materials. List window: You can select one or more transactions and transfer them with the OK button. If you select a transaction by double-clicking on it, the corresponding input window is called up, for example, the input window for a sales order item. In the list, a red dot appears before the quantity if there are still open order requests for this order item. They can be found via the document history. Module name: loggidsp.mod. Classes: CX_TRANSACTION.
