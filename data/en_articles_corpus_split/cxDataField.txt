cxDataField: Data fields belong to the data connectors. While data connectors describe classes, data fields describe the attributes and methods of these classes. Input window: This window is used to maintain data fields. List window: This window is used to list data fields. Selection window: This window is used to select a data field. Access Print Configurator: This window is used to configure access expressions. Access expressions are a description of the navigation through the CyberEnterpriseÂ®. It is navigated from the start object, which is specified as the data origin at the top of the window. On the left side the possible data connectors and their data fields are shown. From here you can navigate further by opening the nodes. It is possible to set a specific data connector as the return type of a method using the context menu. This allows further navigation beyond methods. The selected access paths are displayed on the right side. An access path can be transferred from the tree to this list by double-clicking or drag and drop. Module name: cxDataField.mod. Classes: CX_DATA_FIELD. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages.
