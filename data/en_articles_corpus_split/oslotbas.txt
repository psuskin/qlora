oslotbas. Input window: This module is used to maintain the slots of an object. Selection window: This module serves to list the slots of a business object. Slot base module: The basic module cannot be called up directly. This module serves as the basic module for the modules for maintaining and listing the slots of a business object and is therefore not called directly. All objects derived from the class CX_EXPANDABLE can hold any number of slots, i.e. each of these objects can be extended by the specification of any other attributes (data). Module name: oslotbas.mod. Classes: CX_VESSEL CX_CONTAINER. Module name: custoedt.mod. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages. Module name: oslotsel.mod.
