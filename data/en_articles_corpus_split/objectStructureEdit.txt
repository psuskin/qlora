objectStructureEdit: Easy editing of objects in an ordinal term.Functionality Input window. Input window: In the left-hand list, all nodes/endpoints of the structure of the classification term are displayed as columns. The business objects categorised in all nodes of the structure are listed in such a way that a green tick per column indicates which node (column) this object is assigned to. In the right-hand list, the nodes of the respective business object selected in the right-hand list are displayed. You can add an end point to the selected object here using the "Insert" button or delete it using the "Remove" button. Module name: structureMemberEdit.app. Classes: CX_STRUCTURE.
