atsetobj: Characteristics tables make it easier to work with attributes when you have to assign the same "groups" of attributes to an attribute set again and again. Thus, any attributes can be listed in a characteristics table, so that one only needs to insert the characteristics table. Input window: This window is used to maintain attribute lists (attribute tables). You can select an attribute table by entering the desired number and confirming with RETURN. If you want to open the list window, you can select the menu item "Search". List window: Used to list the characteristics tables. Selection window: This window is used to select a table of characteristics. When the characteristics table is accepted, a dialogue window appears. Here the user can decide whether the characteristics table is to be expanded, whereby the attributes are inserted individually into the attribute set. Module name: cxAttributeSetObject.mod.
