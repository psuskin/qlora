inventoryImport: With this app you can import stock data exported from a materials management system as a file into the ClassiX system. Input window: In this window, all information required for the import is entered and the import is triggered. It is also possible to first check whether the import file can be read in technically and contentwise (e.g. check for units of measure defined in the system). Module name: inventoryImport.app. Classes: CX_INVENTORY.
