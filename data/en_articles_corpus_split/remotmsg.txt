remotmsg: This module is used for communication with other ClassiX® users in the network. For this purpose, either messages can be sent or commands can be executed directly in the remote application. Functionality: When the module is called, the ClassiX® Clients working in the network are collected first: This may take a few seconds, the window title indicates this. Execute command: A command (an InstantView® instruction) can be executed in a remote ClassiX application. Sending a message: A message can be sent to a remote ClassiX application. Module name: remotmsg.mod. Classes: CX_CORBA_MANAGER.
