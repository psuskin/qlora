queryWizard: The Data Query Wizard offers the possibility to quickly display data from the database. The basis for this is provided by data connectors, which also define the possible data fields for querying. A query can be saved permanently as a report. The Data Query Wizard guides you through the steps to create a query, which are executed as tabs in the wizard, only the first of which is mandatory: Select data connector Select columns Select population Select filtering Various examples of how to create a query are presented in a tutorial. Input window: This window guides you through the steps to create a query. Module name: queryWizard.app.
