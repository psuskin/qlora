manufact. Input window: This module is used to maintain the manufacturer data. You can select a manufacturer by entering the manufacturer number in the Number field and confirm with RETURN. The displayed information is divided into 3 areas: master data, article and additional data. Fields marked with "italics" represent data directly from the company. If something is changed in one of these fields, the change is applied to all roles and the company itself. All this data can also be maintained directly in the company input window. List window: This window is used to list the manufacturers. Selection window: This window is used to select a manufacturer. Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Module name: manufedt.mod. Classes: CX_MANUFACTURER. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages. Module name: manufsel.mod. Classes: CX_MANUFACTURER.
