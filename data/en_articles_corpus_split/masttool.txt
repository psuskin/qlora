masttool: Some tools have to be calibrated or checked regularly for their measuring accuracy. For this purpose, so-called test equipment is required to carry out this test or with which the test is performed. These inspection devices are managed and maintenance work is carried out here. The maintenance work due can be determined via the evaluation: Activities -> Production -> Maintenance and processed further from here. Input window: This window is used for maintaining test equipment. List window: This window is used to list the test equipment. Selection window: This window is used to select test equipment. Option window: All settings for the search can be made here. The window can only be closed with the "OK" button. Module name: "masttool".mod. Classes: CX_MASTER_TOOL. Security: In addition to restricting access rights via the class and its data fields, the module can be restricted in its use via some of the received messages. Module name: masttedt.mod.
