dtausedt: The paperless exchange of data media with banks for domestic payments (Germany) is carried out using the DTAUS file format. This module is suitable for reading in a file of this format and displaying the data in an edited form. Analyse DTAUS file: The purpose of reading in a file in DTAUS format is to display the contents of the file in an edited form and to check the error-free structure of the file. Module name: dtausedt.mod. Classes: CX_ASCII_FILE.
