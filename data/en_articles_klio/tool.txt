This is the description of the module "tool" with the name "Tool (module)": Tools are defined in the CyberEnterprise® enterprise model as a part role. This means that tools can also have other parts roles, such as order or production roles, or be managed as assets in the asset master. They can also be kept in stock. The part assigned to a tool is called equipment and is also managed separately in the parts master. In contrast, in the CyberEnterprise® Enterprise Model, machines and vehicles(cars and trucks) are considered as special characteristics of parts themselves. This is the description of the functionality of the module "tool" with the name "Tool (module)" regarding Input window: This window is used to maintain tools. You can select a tool by entering the desired number in the Number field and confirming with RETURN. If you want to call up the list window, you can select the menu item "Search". This is the description of the functionality of the module "tool" with the name "Tool (module)" regarding List window: Serves to list the tools. This is the description of the functionality of the module "tool" with the name "Tool (module)" regarding Selection window: This window is used to select a tool. This is the description of the functionality of the module "tool" with the name "Tool (module)" regarding Option window: All settings for the search can be made here. The window can only be closed with the "OK" button.
