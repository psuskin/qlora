This is the description of the module "slot" with the name "Dynamic data fields (slots)": With this application you can create new dynamic data fields (slots). You can also display the objects that use a specific dynamic data field from this app (see menu item "Use"). This is the description of the functionality of the module "slot" with the name "Dynamic data fields (slots)" regarding List window: This list shows all dynamic data fields available in the system. This is the description of the functionality of the module "slot" with the name "Dynamic data fields (slots)" regarding Selection window: This window is used to select a dynamic data field.
