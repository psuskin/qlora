This is the description of the functionality of the module "loggipor" with the name "Order status" regarding Description: On the one hand, this app is used to check/display order items in a certain status (e.g. to evaluate the order commitment), but it is also used to select order items in order to transfer them to the goods receipts, for example. Accordingly, items from supplier and/or customer complaints can also be included in the evaluation. Both document types can lead to a goods receipt and can therefore be included here. This is the description of the functionality of the module "loggipor" with the name "Order status" regarding Functionality: The order items recorded in the system can be listed according to a wide variety of criteria.
