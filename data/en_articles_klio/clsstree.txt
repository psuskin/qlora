This is the description of the module "clsstree" with the name "Class tree": This app displays the classes of the CyberEnterprise® enterprise model in a tree structure. If a class with a blue diamond is displayed, then it is a class defined in C++, if the colour of the diamond is red, then it is a pseudo class, i.e. a class without own definition in C++. This is the description of the functionality of the module "clsstree" with the name "Class tree" regarding Class tree window: This window displays the classes of the CyberEnterprise® Enterprise Model. This is the description of the functionality of the module "clsstree" with the name "Class tree" regarding Selection window: This window is used to select a class. The class can be selected by double-clicking on it. By using the right mouse button you can switch to the tree view.
