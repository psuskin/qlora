This is the description of the functionality of the module "Pythia_legalPersonList_host" with the name "Company list Host" regarding List window: This window is used to list the companies. If a company has several addresses, telephone or fax numbers, only the first data is displayed in the company list. On the right of the window is a list of the roles and communication objects of the currently selected company. Communication objects can be edited with a double click. Double-clicking on a role takes you directly to its input window. With a right click, the context menu can be called up to create new roles or to display the validity.
