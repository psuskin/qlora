This is the description of the module "represen" with the name "Representative": Description Functionality Related topics Technical documentation. This is the description of the functionality of the module "represen" with the name "Representative" regarding Description: Within the CyberEnterprise business OS, the partner function "Representative" represents persons or companies who work for you in sales. Representatives can therefore also be commission recipients, the latter term is used synonymously within the CyberEnterprise business OS. This is the description of the functionality of the module "represen" with the name "Representative" regarding Input window: This module is used to maintain the substitute data. You can select a substitute by entering the substitute number in the Number field and confirming with RETURN. If you want to call up the substitute list, you can either double-click the "Number" field or select the menu item "Search". Fields marked with "italics" represent data directly from the company. If something is changed in one of these fields, the change is applied to all roles and the company itself. All this data can also be maintained directly in the company input window. This is the description of the functionality of the module "represen" with the name "Representative" regarding List window: This window is used to list the representatives. This is the description of the functionality of the module "represen" with the name "Representative" regarding Selection window: This window is used to select a substitute. This is the description of the functionality of the module "represen" with the name "Representative" regarding Option window: All settings for the search can be made here. The window can only be closed with the "OK" button.
